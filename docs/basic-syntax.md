# 基本構文

## コア

```
; ============================================================
; コア機能
; ============================================================

; 定義
(def name value)

; 関数定義
(def (fname arg1 arg2 ...)
  expr)

; 束縛
(let (name1 expr1
      name2 expr2
      ...)
  body)

; パターンマッチング
(match value
  pattern1 expr1
  pattern2 expr2
  ...)

; if式
(if pred
  then-expr
  else-epxr)

; 条件分岐
(cond
  pred1 expr1
  pred2 expr2
  ...
  else  exprN)

; 無名関数
(fn (arg1 arg2 ...)
  body)

; 非モナディックな逐次実行
(begin
  expr1
  expr2
  ...
  result)
```

## 演算子の表層形式

```
; ============================================================
; 演算子の表層形式
; ============================================================

; 連鎖形式
; (a op b op c op ...)

; 例
; (a * b * c)
; (x |> f |> g)
```

## モナドスタイル

```
; ============================================================
; ある文脈における逐次的な集約や変換処理(モナドスタイル。Haskellのdo記法をほぼそのまま輸入)
; ============================================================

(do
  (a <- ma)
  (n = x)
  expr)
```

## アプリカティブファンクタスタイル

```
; ============================================================
; ある文脈における並列的な集約や変換処理(アプリカティブファンクタ的なスタイルの構文。OCamlのlet+を参考にした。命名は、値を取り出しているという点で、PythonやElixirのwithの動きに似ているため)
; ============================================================

(with (a aa
       b ab
       c ac)
  (f a b c))
```
