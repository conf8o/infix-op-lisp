# 型宣言

`(:)` も型宣言を行うための演算子ととらえ、前置スタイル、中置スタイルで型宣言できます。

```
; ============================================================
; 型宣言
; ============================================================

; --- 前置スタイル ---
(: x Int)
(: f (Int -> Int))
(: (+)  (Int -> Int -> Int))

; --- 中置スタイル(同じ情報、異なる表層形式) ---
(x : Int)
(f : (Int -> Int))
((+) : (Int -> Int -> Int))
```
