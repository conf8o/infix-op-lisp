# infix-op-lisp

中置演算子を標準でサポートするLisp系関数型言語の仕様書です。Clojure, Dr.Racket, Haskell, Elm, OCaml, をベースに、シンプルで一貫性のある構文を提供します。

## 基本理念

- 糖衣構文や表層形式には常に正規形があること
- 表層形式と正規形を明確に区別すること
- 演算子優先順位を持たず、明示的な括弧を要求すること
- シンプルな構文であること（わずらわしさが少ないこと）
- 一貫性とわかりやすさを両立すること
- 関数型プログラミングができること
